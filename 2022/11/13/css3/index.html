盒子模型及边框的使用

盒子模型有外边矩margin，边框border和内边框padding，可以使用css来控制这三个属性，而最里面的content是html的内容。

默认情况下设置的width和height只是在设定里面的content的内容，而很多标签如已经自带了margin等属性，故可以通过margin：0；来使得外边框不起作用。

盒子的宽度 = 内容宽度 + 左填充 + 右填充 + 左边框 + 右边框 + 左边距 + 右边距
盒子的高度 = 内容高度 + 上填充 + 下填充 + 上边框 + 下边框 + 上边距 + 下边距

margin-top：上边距
margin-buttom：下边距
margin-left：左边距
margin-right：右边距

可以单独设置每一条边，也可以直接用一个margin来统一设置四条边。

margin：0，auto；实现了水平居中，但auto无法实现垂直居中，因为水平方向上自动填充了父元素，即盒子单独占一行，而竖直方向上没有。

padding设置内边框也是如此，但内边框与外边框不同，内边框可以设置背景颜色和图片。

border 1px solid red dashed表示虚线

浮动

块级元素：独占一行

行内元素：不独占一行

行内元素可以被包含在块级元素里面

使用display：inline-block 使得即在行内又有块的属性

定位

1.相对定位：position：relative；

相对于原来的位置进行指定的偏移，仍然在标准文档流中，原来的位置会被保留

2.绝对定位：position：absolute

如果有父级元素，则基于最近的已定位的父级元素的位置进行绝对定位。

如果没有，则基于boby或者html即浏览器的边框进行绝对定位。

3.固定定位：position：fixed

元素的位置相对于浏览器窗口是固定位置，即使窗口滚动它也不会移动。

页面中某些元素需要一直出现在某个地方。

4.z-index调整层级，默认是负一，越大越优先。
